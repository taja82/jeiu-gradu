이것은 인천재능대학교 캡스톤디자인을 위해 제작된 웹 페이지입니다.
node.js 기반으로 작성하였으며, node.js와 npm, 각종 모듈이 설치되어 있지 않다면 제대로 실행이 되지 않을 수 있습니다.
여기에는 오늘 뭐 먹지?(카카오맵api를 기반으로 만든 웹페이지) 뿐만 아니라, 홍보 영상에 잠깐 사용하였던 블로그 소스(socket.io를 이용하여 여러 사용자가 채팅처럼 이용할 수 있는 웹페이지),
테스트로 사용한 여러 페이지들이 포함되어 있습니다.
오늘 뭐 먹지 사이트는 express 모듈 라우팅 기능을 이용하여, bootstsrap이나, jquery를 로드하고, 원하는 주소에 표시될 수 있도록 적용한 것 말고는, 파일 내부에는 javascript, jquery로 작성하였습니다.


06-20일 추가사항

몇몇 항목들은 입력하지 않았기 때문에 내용 추가합니다.

Node.js 와 npm 설치 이후 현재 디렉토리에서 npm start 명령어로 실행 시킬 수 있으며, 127.0.0.1:3000/map이나 localhost:3000/map 주소로 통하여 결과를 확인해 볼 수 있습니다.
또한 https://jeiu-gradu.herokuapp.com/map 이 주소를 통하여 이미 올려놓았던 결과를 확인해 볼수도 있습니다.

현 사이트 제작은 아직 미완성인 상태이므로, 몇몇 기능이 제대로 동작하지 않을 수 있고, db연결이 "전혀" 되어 있지 않기 때문에 페이지를 새로고침을 하게 되면 모든게 초기화 되게 됩니다.
카카오맵과 카카오로그인을 포함한 각종 소셜로그인을 위해, 각 사이트 api를 사용하였습니다.

라이브러리를 제외한 대부분은 저 권석진이 직접 만들었으며, 무단배포를 금지합니다.
